Step 4: producer.py
Connects to Redis server
Generates fake users, fake content, and engagement
Combines everything into a single post dictionary
Pushes he post to a Redis streams
Repeats every second

Step 5: consumer.py
Connect to Redis server
Read social_posts stream from ID "0"
Wait for new posts using XREAD
For each received post, call update_summary(post) to update hashtag stats
Print current aggregated summary
Update last_id so next xread fetches only newer posts
Reapeat forever

Step 6: window_aggregator.py
Creating a dictionary for every different # using defaultdict
a function that stores posts, likes ,comments, shares for each # (current_window)
for loop creates iterations for one # at a time
row = {..} creates one record/one row in ML dataset
append all rows to windows_history, clear current_window to start the next time window
export windows_history to csv using pd

Phase 3: feature_engineering.py
Creating features like time and lag feature to record activity

Phase 4: forecast.py
Training model to make forecasts
Import dataset, converting time to required datatype
Looping through every unique hashtag
Renaming columns to Prophets requirements
Choose big enough datasets (>10 cols)
Train Prophet model
Create future timestamp for next 30 mins
Train model, a new model for each #, since each # has a different behavior
Combining forecasts, save and print

Phase 5: analytics.py
Trend Ranking + Anomaly Detection
Loading data
Convert timestamps into dateframe format
Select the latest record for each hashtag.
Select the first future prediction for each hashtag.
Merge current data with forecast data.
Fill missing forecast values with 0.
Compute the forecast error (actual posts − predicted posts).
Detect anomalies when the error exceeds 30% of the prediction.
Normalize important features using Min-Max Scaling.
Calculate a weighted Trend Rank Score.
Classify each hashtag as Rising, Falling, or Stable.
Rank hashtags from highest to lowest trend score.
Round numeric values for readability.
Save the final results to data/insights.csv and print the ranked hashtags.
